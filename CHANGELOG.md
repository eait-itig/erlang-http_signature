# Changelog

## 1.1.0 (2015-07-18)

* Enhancements
  * Elixir implementation.

* Bug Fixes
  * Full import/export of ecdsa public keys.

## 1.0.0 (2015-07-17)

* Enhancements
  * Initial release.
  * Supports the following algorithms
    * `dsa-sha1`
    * `ecdsa-sha1`
    * `ecdsa-sha256`
    * `ecdsa-sha512`
    * `hmac-sha1`
    * `hmac-sha256`
    * `hmac-sha512`
    * `rsa-sha1`
    * `rsa-sha256`
    * `rsa-sha512`
